## 一、核心参考（必读）

### 1. jskyzero《怪物猎人中的 AI 出招设计》
- **链接**：https://zhuanlan.zhihu.com/p/645269557
- **作者主页（设计者笔记）**：https://jskyzero.github.io/
- **视角**：拆包 + 体验反推双视角（MHRSB 数据）
- **重点摘要**：
  - **三个核心结论**（结构层面，不涉及具体招式）：
    1. **出招跟状态相关**：普通/愤怒/疲劳/特殊强化 各自一套派生
    2. **出招满足反应性 + 随机性**：条件 → 选择池 → 随机
    3. **出招存在固定派生**：可背板，但中途可调整/取消
  - **拆包视角关键发现**：
    - 雌火龙（em001-00）的 `ai_fsm_user_data` 下有 **125 个 .user.2 文件**
    - 文件按场景细分：hub / damage / die / fly / think 等
    - 底层驱动技术：**有限状态机（FSM）**
    - 设计原则：把复杂逻辑拆分到不同场合的单个逻辑里，单个逻辑文件可作为"特性组件"动态挂载/卸载
    - 横向对比 em001（雌火龙）/ em093（爵银龙）文件结构，确认"特性组件化"思路
- **价值**：本文是 MH 怪物 AI 拆解的理论骨架来源

### 2. 游民星空《怪物猎人荒野全大型怪物图鉴及招式解析》
- **链接**：https://www.gamersky.com/z/monster-hunter-wild/1681755_37580/
- **作者**：Wioud
- **发布时间**：2025-02-28（首发版）
- **结构**（每个怪物）：
  - 怪物信息（栖息地、弱点、可破坏部位、素材掉落概率）
  - **招式应对方法**（每招描述：前摇动作 → 招式过程 → 玩家应对）
- **重点示例**：缠蛙的招式应对表
  - 右前爪击：前摇 → 抬手蓄力 → 砸地 → 击飞效果 → 有充足躲避时间
  - 啃咬及舌头转圈：前摇 → 吐舌头张嘴 → 啃一口 → 舌头旋转一圈
  - 冲刺：前摇 → 蓄力 → 持续伤害
  - 跳跃砸地：前摇 → 摇屁股蓄力 → 跳半空 → 砸下
- **价值**：每个招式的"前摇 → 判定 → 后摇 → 应对"结构，可直接作为模板的招式库参考

## 三、模板设计原则

- **拆解主轴**：敌人 AI（用户自评天赋点）
- 不泛拆"战斗系统"，聚焦"怪物作为 AI 是怎么决策的"
- 每只怪物独立成档，**但所有怪物共用同一套模板结构**（横向可比）
- 所有具体招式/帧数/数值 → 用户填答，Mavis 不主动编造
- 主拆解对象：**MH 荒野**（当前最新且最复杂）

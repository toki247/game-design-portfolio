## 1. Game Overview(游戏概述)

### 1.1 Game Concept(游戏概念)✅ 已定

一个 RPG,玩家扮演被洗脑的魔王军封印师"勇者",**最强开局,越打越弱**,6 场 boss 战揭示"国王才是真凶"的真相,最终在国王战中觉醒、收回能力、击败国王。

核心反常识点:**整条成长曲线倒过来画**。

### 1.2 Feature Set(特性集)✅ 已定

- **失去 = trade-off + 觉醒的代价**:每次使用封印术失去一样东西,但获得洞察力 + 削弱洗脑魔法
- **6 场 boss 战各用不同解法**:ABCE 四种 + 综合
- **双线塑造世界观**:战斗对话(主角 ↔ 魔族)+ 休息对话(主角 ↔ 小精灵)
- **多结局**:悲剧结局(失去生命)vs 救赎结局(觉醒 + 收回能力 + 击败国王)
- **篝火夜话休息场景**:与小精灵的友情 + 记忆浮现
- **ACT 系统**(参考 Undertale):每场战斗的"特殊解法"通过独特 ACT 触发

### 1.3 Genre(类型)✅ 已定

回合制 RPG(2D),后调整为 **2D 类塞尔达玩法**(具体玩法待定)

参考作品:
- Undertale(ACT 系统、战斗特殊解法)
- 塞尔达 缩小帽 / 梦见岛(2D 动作冒险)
- 哈迪斯 2(作品集对标)
- 碧蓝幻想 Relink(作品集对标)

### 1.5 Game Flow Summary(游戏流程概述)✅ 已定

11 个场景的线性流程:

```
开场(最强状态演示 + 国王交付任务)
  ↓
休息 1(篝火夜话 - 与小精灵初遇)
  ↓
战斗 1(用【技能】战胜 → 失去技能)
  ↓ 休息 2 → 战斗 2(用【装备】战胜 → 失去装备)
  ↓ 休息 3 → 战斗 3(用【等级】战胜 → 失去等级)
  ↓ 休息 4 → 战斗 4(用【珍贵道具】战胜 → 失去珍贵道具)
  ↓ 休息 5(最长的一次,主角开始觉醒)
  ↓
战斗 5(魔王 - 综合解法)
  ├ 失去生命 → 悲剧结局
  └ 揭示真相 → 真结局 → 转国王战
战斗 6(国王 - 收回能力,击败国王 → 救赎结局)
```

## 2. Gameplay and Mechanics(玩法与机制)

### 2.1 Game Progression(游戏进程)✅ 已定

线性流程,无分支(直到魔王战才出现多结局):
- 主线:开场 → 5 休息 + 4 天王战 → 魔王战 → (分支)→ 国王战
- 支线:无
- 难度:无主动调节(由"失去"自然形成)

### 2.2 Core Mechanic - 失去的 trade-off(核心机制)✅ 已定

**这是整个游戏的设计灵魂。**

每次使用封印术:
- 失去战斗力(技能 / 装备 / 等级 / 珍贵道具)
- **同时获得**:
  - 解开下一关的钥匙(洞察力)
  - 削弱洗脑魔法 → 主角逐渐恢复记忆

**机制层面**:玩家用"战斗力"换"洞察力"
**叙事层面**:主角在战斗中逐渐觉醒
**形式即内容**:整条"递减曲线"本身就是叙事结构

### 2.5 ACT 系统 ✅ 已定(框架)

每场战斗的"关键" = 主角当前拥有的 4 样东西之一。
玩家通过独特的 ACT 选项"发现"哪样东西是关键 → 用关键东西战胜 → 战后失去它 + 获得洞察力。

参考 Undertale 的 ACT 系统,每场战斗的 ACT 是独特的。

**核心循环**:
```
[每场战斗前] 主角拥有 X 样东西(开局 4 样,过程中递减)
  ↓
[战斗开始] 玩家不知道哪样东西是关键
  ↓
[独特的 ACT 选项] 让玩家"发现"哪样东西是关键
  ↓
[玩家选择] 用关键东西战胜敌人
  ↓
[战后] 失去那件东西 + 获得洞察力
  ↓
[下一关] 用剩余的东西 + 上一关的洞察 → 战胜下一个敌人
```

## 3. Story, Setting and Character(故事、设定与角色)

### 3.1 Story and Narrative(故事与叙事)✅ 已定

#### 3.1.1 故事梗概

主角原本是魔王军的封印师,擅长"献出一样东西封印敌人"的招式。国王想除掉魔族(因魔族拥有某种资源/力量)——但魔族难杀,只能封印。国王抓来主角,洗脑成"勇者",派他封印四天王和魔王。

主角在旅途中遇到小精灵,结伴同行。每场四天王战用不同解法,战后失去一样东西 + 获得洞察力。每段休息篝火夜话,友情加深 + 主角零星记忆浮现。

随着封印术的使用,洗脑魔法逐渐失效。主角通过战斗对话(线 1)和休息对话(线 2),逐渐意识到魔族不是坏人、国王才是真凶。

魔王战(综合解法),主角几乎失去所有。最后两个选择:
- 失去生命 → 悲剧结局
- 揭示真相,停止魔王战 → 真结局 → 转国王战

国王战,小精灵"拉出迷惘",主角完全恢复记忆,收回失去的能力,用全盛状态击败国王 → 救赎结局。

#### 3.1.2 叙事结构(多幕剧)

| 阶段 | 场景 | 基调 |
|---|---|---|
| 第一幕 | 开场 ~ 战斗 2 | 轻松搞怪 + 友情培养 |
| 第二幕 | 战斗 3 ~ 4 | 觉醒 + 态度转变 |
| 第三幕 | 魔王 ~ 国王 | 悲壮 + 救赎 |

#### 3.1.3 叙事 vs 玩法融合

- **失去** = 机制 trade-off + 叙事觉醒的代价(同时发生)
- **ACT** = 玩家在战斗中"理解"敌人 → 叙事上发现魔族不是坏人
- **篝火夜话** = 友情 + 记忆浮现,跟战斗的"理解"形成对照
- **多结局** = 玩家的选择决定叙事走向

#### 3.1.4 双线塑造

| 场景 | 线 1 | 线 2 |
|---|---|---|
| 战斗 | 主角 ↔ 魔族(boss) | boss 攻击模式暗示真相 |
| 休息 | — | 主角 ↔ 小精灵 |

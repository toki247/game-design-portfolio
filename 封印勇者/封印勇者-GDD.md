## 1. Game Overview(游戏概述)

### 1.1 Game Concept(游戏概念)✅ 已定

一个 RPG,玩家扮演被洗脑的魔王军封印师"勇者",**最强开局,越打越弱**,6 场 boss 战揭示"国王才是真凶"的真相,最终在国王战中觉醒、收回能力、击败国王。

核心反常识点:**整条成长曲线倒过来画**。

### 1.2 Feature Set(特性集)✅ 已定(2026-07-01)

**核心机制**:
- **5 能力**:圣剑 / 盾牌 / 弓箭 / 护身符 / 生命(Max HP 5)
- **boss 战中只解锁对策能力 + 闪避**:其他能力禁用,强制专注
- **线性强化链**:boss 战后消耗对策能力,强化下一个 boss 的对策能力
  - Boss 1 魔法师 → 护身符 → 强化弓箭
  - Boss 2 骑士 → 强化弓箭 → 强化盾牌
  - Boss 3 剑士 → 强化盾牌 → 强化圣剑
  - Boss 4 进阶剑士 → 强化圣剑
- **魔王战**:无能力 + 封印术消耗 Max HP(5→4→3→2→1)
- **多结局**:完全封印(再消耗 1)→ 悲剧 / 不完全封印 → 真结局 → 国王战
- **国王战**:5 能力觉醒 + 综合考验 → 救赎结局

**游戏类型**:
- 2D 类塞尔达动作(45 度俯视,Camera 不动 + Sprite 旋转 45 度)
- 即时战斗(无回合制,无 ACT 菜单)
- boss 战的解法通过**操作手感**呈现(弹反、蓄能、闪避、护身符破解黑雾等)
- 篝火夜话休息场景(对话界面,跟小精灵聊天 + 记忆片段)

## 2. Gameplay and Mechanics(玩法与机制)

### 2.1 Game Progression(游戏进程)✅ 已定

线性流程,无分支(直到魔王战才出现多结局):
- 主线:开场 → 5 休息 + 4 天王战 → 魔王战 → (分支)→ 国王战
- 支线:无
- 难度:无主动调节(由"失去"自然形成)

### 2.2 Core Mechanic - 5 能力 + 强化链 + Max HP ✅ 已定(2026-06-30)

**这是整个游戏的设计灵魂。**

主角拥有 **5 种能力**:圣剑 / 盾牌 / 弓箭 / 护身符 / 生命(Max HP 5 点)。

每场 boss 战中,**只解锁对策能力 + 闪避**,其他能力禁用——强制专注。

#### 2.2.1 能力强化链(线性,叙事驱动)

每个 boss 战后,**消耗该 boss 的对策能力,让下一个 boss 的对策能力获得强化**:

```
Boss 1 魔法师 → 对策:护身符(释放光亮破解黑雾)
  战后:护身符消耗 → 弓箭获得"魔法附着"(能射穿铠甲)

Boss 2 骑士 → 对策:强化弓箭(魔法附着 + 射眼睛)
  战后:弓箭消耗 → 盾牌获得"弹反能力"(能扛剑士攻击)

Boss 3 剑士 → 对策:强化盾牌(弹反 + 输出窗口)
  战后:盾牌消耗 → 圣剑获得"蓄能能力"(能蓄能攻击)

Boss 4 进阶剑士(比 Boss 3 更强)→ 对策:强化圣剑(蓄能攻击)
  战后:圣剑消耗
```

**线性强化链**:护身符 → 弓箭 → 盾牌 → 圣剑。每个 boss 战消耗一个能力,**强化**下一个 boss 的对策能力。

#### 2.2.2 生命系统(Max HP 消耗)

- **初始 Max HP**:5 点
- **魔王战机制**:每次封印消耗 1 点 Max HP,造成魔王重创
- **总封印次数**:4 次(消耗 4 点 → Max HP 从 5 → 1,进入最终 CG)
- **最后 1 点 Max HP 是"选择"**:
  - 完全封印(再消耗 1 点)→ 主角耗尽所有 → **悲剧结局**
  - 不完全封印 → 保留 1 点 → **真结局** → 进入国王战
- **消耗 Max HP 而非当前 HP**——每次封印都是永久降低,玩家能"感到自己要死了"

#### 2.2.3 终局能力回归(觉醒)

- **国王战**:5 能力全部回归 + 综合考验
- 这是"觉醒"的物化呈现——主角作为封印师的所有力量收回

**机制层面**:能力不是简单消失,而是**线性传承**(boss 的力量换宿主)
**叙事层面**:每次封印牺牲 Max HP,直观体现"封印术的代价"
**形式即内容**:整条"递减 → 强化 → 觉醒"曲线本身就是叙事结构

### 2.5 战斗循环 ✅ 已定(2026-07-01)

**核心循环**:
```
[每场战斗前] 主角拥有部分 5 能力(开场 5 样,boss 战中只解锁 1 个对策能力)
  ↓
[boss 战中] 只解锁当前 boss 的对策能力 + 闪避(其他能力禁用)
  ↓
[玩家使用对策能力] 战胜 boss(通过操作手感,不需要 ACT 选项)
  ↓
[战后] 对策能力消耗 → 强化下一个 boss 的对策能力
  ↓
[下一关] 用强化后的能力 + 闪避 → 战胜下一个 boss
```

**设计要点**:
- **不需要 ACT 菜单选项**——boss 战的解法直接通过操作呈现(弹反、蓄能、闪避、护身符破解黑雾等)
- boss 战中**只解锁对策能力 + 闪避**:其他能力禁用,强制专注
- **强化是叙事性的**:boss 战后,主角把当前能力的力量"传承"给下一个能力
- 操作手感的差异化是 boss 战的核心乐趣

### 2.6 封印术与失去 ✅ 已定(2026-06-30 更新)

**基础设定**(叙事):
- 魔族不能杀只能封印——这是主角作为封印师的核心设定
- 每次使用封印术需要**献出一样东西**作为代价

**机制层面**(2026-06-30 重设计):
- **boss 战中**:boss 对策能力用完后被"消耗"——这是封印术的代价
- **魔王战**:每次封印消耗 1 点 Max HP
- **5 能力的消耗顺序**(线性强化链):
  1. Boss 1 战后:护身符消耗(强化弓箭)
  2. Boss 2 战后:弓箭消耗(强化盾牌)
  3. Boss 3 战后:盾牌消耗(强化圣剑)
  4. Boss 4 战后:圣剑消耗
  5. 魔王战:消耗 Max HP 4 次
- **消耗的"叙事含义"**:不是单纯失去,而是"boss 的力量转移到下一个能力上"

**失去的呈现**:
- 战后有过场动画(具体设计待定)
- 主角/敌人/boss 三方反应(具体设计待定)
- 主角台词:"我把它(护身符)的力量注入了弓箭..."——叙事性描述

## 3. Story, Setting and Character(故事、设定与角色)

### 3.1 Story and Narrative(故事与叙事)✅ 已定

#### 3.1.1 故事梗概

主角原本是魔王军的封印师,擅长"献出一样东西封印敌人"的招式。国王想除掉魔族(因魔族拥有某种资源/力量)——但魔族难杀,只能封印。国王抓来主角,洗脑成"勇者",派他封印四天王和魔王。

主角在旅途中遇到小精灵,结伴同行。每场四天王战用对应能力,战后失去该能力 + 强化下一个 boss 的对策能力。每段休息篝火夜话,友情加深 + 主角零星记忆浮现。

随着封印术的使用,洗脑魔法逐渐失效。主角通过战斗对话(线 1)和休息对话(线 2),逐渐意识到魔族不是坏人、国王才是真凶。

魔王战(综合解法),主角几乎失去所有。最后两个选择:
- 失去生命 → 悲剧结局
- 揭示真相,停止魔王战 → 真结局 → 转国王战

国王战,小精灵"拉出迷惘",主角完全恢复记忆,收回失去的能力,用全盛状态击败国王 → 救赎结局。

#### 3.1.2 叙事结构(多幕剧)

| 阶段 | 场景 | 基调 |
|---|---|---|
| 第一幕 | 开场 ~ 战斗 2 | 轻松搞怪 + 友情培养 |
| 第二幕 | 战斗 3 ~ 4 | 觉醒 + 态度转变 |
| 第三幕 | 魔王 ~ 国王 | 悲壮 + 救赎 |

#### 3.1.3 叙事 vs 玩法融合

- **失去** = 机制 trade-off + 叙事觉醒的代价(同时发生)
- **ACT** = 玩家在战斗中"理解"敌人 → 叙事上发现魔族不是坏人
- **篝火夜话** = 友情 + 记忆浮现,跟战斗的"理解"形成对照
- **多结局** = 玩家的选择决定叙事走向

### 4.4 Boss 详细设定 ✅ 已定(2026-06-30)

#### Boss 1:魔法师

- **类型**:四天王之一,擅长范围控制
- **特性**:黑雾遮蔽视野,降低玩家视觉范围
- **对策**:**护身符**——释放光亮破解黑雾,恢复视野
- **ACT 设计**:
  - 让玩家发现"光亮能破解黑雾"(对话/观察/记忆)
  - 护身符在 boss 战中作为对策能力解锁
- **战斗流程**:
  - 默认状态:玩家视野受限,只能用护身符解除黑雾
  - 解除后:玩家视野恢复,可以正常攻击 boss
- **战后**:护身符消耗 → 弓箭获得"魔法附着"(强化下一次 boss 战)
- **叙事含义**:护身符的"光"属性转移到弓箭上,让弓箭获得魔法属性

#### Boss 2:骑士

- **类型**:四天王之一,擅长高速冲击
- **特性**:冲撞 + 重铠甲(普通攻击无法穿透)
- **对策**:**强化弓箭**——魔法附着 + 抓住时机射穿眼睛
- **ACT 设计**:
  - 抓冲撞的间隙(玩家观察 boss 的攻击模式)
  - 用魔法弓箭瞄准暴露的眼睛造成致命伤害
- **战斗流程**:
  - boss 冲撞 → 玩家闪避 → 观察时机 → 强化弓箭射眼
- **战后**:弓箭消耗 → 盾牌获得"弹反能力"(强化下一次 boss 战)
- **叙事含义**:弓箭的"精准穿透"转移到盾牌上,让盾牌能反弹剑士的攻击

#### Boss 3:剑士

- **类型**:四天王之一,擅长近战攻击
- **特性**:连续剑击,需要抓住输出窗口
- **对策**:**强化盾牌**——弹反攻击 → 获得输出窗口
- **ACT 设计**:
  - 玩家在 boss 攻击的时机按"弹反"键
  - 成功弹反 → boss 短暂硬直 → 玩家输出窗口
- **战斗流程**:
  - boss 连续剑击 → 玩家举盾 → 弹反成功 → 输出窗口 → 攻击
- **战后**:盾牌消耗 → 圣剑获得"蓄能能力"(强化下一次 boss 战)
- **叙事含义**:盾牌的"反射"属性转移到圣剑上,让圣剑能蓄能攻击

#### Boss 4:进阶剑士

- **类型**:四天王之首 / 剑圣
- **特性**:比 Boss 3 更强,攻击更猛更难弹(因为 Boss 3 已经用过了)
- **对策**:**强化圣剑**——蓄能攻击(只有闪避 + 蓄能剑击)
- **ACT 设计**:
  - 玩家不能弹反(只是闪避),只能蓄能圣剑
  - 蓄能需要时间,蓄能期间被击中就断蓄
- **战斗流程**:
  - boss 强力攻击 → 玩家闪避 → 找空隙蓄能 → 释放蓄能攻击
- **战后**:圣剑消耗
- **叙事含义**:圣剑的"蓄能"是 Boss 3 盾牌的反射属性升级而来

#### Boss 5:魔王(综合战)

- **类型**:最终 boss
- **特性**:综合 4 boss 的挑战(黑雾 + 冲撞 + 剑击 + 远程)
- **玩家能力**:**无能力**,只有闪避 + 封印术 + 5 点 Max HP
- **封印术机制**:每次封印消耗 1 Max HP,造成魔王重创
- **总封印次数**:4 次(Max HP 5 → 1)
- **多结局分支**(最后 1 Max HP 的选择):
  - **完全封印**(再消耗 1 点)→ 主角耗尽所有 → **悲剧结局**(进入最终 CG:主角封印魔王后倒下)
  - **不完全封印** → 保留 1 Max HP → 进入**真结局** → 国王战
- **叙事引导**:通过叙事让玩家意识到"不该完全封印"(具体方式待细化)
- **战后**:无论哪个结局,主角都进入最终 CG(进入角色发展弧的终结)

#### Boss 6:国王(真结局战)

- **类型**:隐藏 boss(真结局专属)
- **特性**:魔王战后的"真正的敌人"
- **战斗能力**:无固定模式(根据玩家选择动态调整)
- **玩家能力**:**5 能力全部回归**(觉醒状态)
- **觉醒机制**:小精灵"拉出迷惘" → 主角恢复记忆 → 5 能力(圣剑/盾牌/弓箭/护身符/生命)全部回来
- **战斗流程**:
  - 开场:小精灵揭示主角真实身份(被洗脑的封印师)
  - 主角觉醒 → 5 能力回归
  - 综合考验:用所有能力击败国王
- **战后结局**:**救赎结局**——主角恢复记忆,击败真凶,获得平静
